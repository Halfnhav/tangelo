A simple web framework for big data.

Tangelo makes it easy to create modern, dynamic web applications. With Tangelo, you can use a built-in data service, or write simple Python functions to access serve your data. Our client library, tangelo.js, wraps charts, maps, and other advanced HTML5 visualizations for fresh and dynamic applications with minimal code.

Built for ease of development

Starting a Tangelo application is as simple as creating an index.html file, which can refer to supporting Javascript and CSS as needed. If you need custom server-side processing, data gathering, or a full REST API, put a Python file right alongside your Javascript and HTML to expose it as a service.

For example, suppose you want to manipulate some text on your server. The app starts with simple Bootstrap-infused HTML file containing a form to submit the text.

The Javascript code provides a handler for the button to submit the text to the server and display the results. Note that we are using a custom service called myservice.

To implement a service, just add a Python file anywhere. It will automatically be exposed as a URL endpoint by the name of the Python file, here as myservice. In Python, define a run() function with arbitrary arguments, and URL paths and/or query parameters will be mapped to those arguments. If you return a JSON-like Python object (arbitrarily-nested dictionaries/arrays), Tangelo will convert that to JSON text so it can be retrieved easily using Javascript Ajax utilities like jQuery's $.getJSON(). Here we simply return the reversed text.

That's it. Add these three files under the build/deploy/web directory of your Tangelo instance, and the app will start being served on Tangelo. If you edit any of these files, including the Python file, just refresh your page in the browser to see the changes, no web server restart required.
